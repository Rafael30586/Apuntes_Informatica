Supongamos que tenemos unas variables que declaramos así:

int variable = 10;
int *puntero = &variable;

Podemos imprimir los siguientes valores:

printf("%p", puntero) --> imprime la dirección de memoria que almacena el puntero
printf("%d", *puntero) --> imprime el valor al que apunta el puntero, que en este caso sería 10. A este procedimiento se lo llama en inglés "dereference"

--------------------
Si a una puntero que apunta a un array se le suma un número entero (como 2 o 3 por ejemplo), el puntero se moverá dicha cantidad de posiciones. Ejemplo:

int numeros = {30,12,95,61,90,27};
int *puntero = números;
printf("%d",*puntero); // imprime 30
puntero = puntero + 2;
printf("%d",*puntero); // imprime 95

Cuando le sumamos o restamos números enteros a un puntero que apunta a un valor de un array, este puntero se mueve posiciones dentro del array, no importa de qué tipo sea, si el array es de char, de int, de double, etc. En un array de char, el puntero se mueve 1 posición de memoria, en un array de int, el puntero se mueve 4 posiciones de memoria, etc

---------------------
Si tenemos dos punteros, uno que apunta a la posición 1 de un array y otro que apunta a la posición 4 del mismo array, podemos restarlos para que nos de como resultado la distancia en elementos que hay entre un puntero y el otro

---------------------

Podemos crear punteros que apunten hacia otro puntero, por ejemplo:

int numero = 10;
int *puntero1 = &numero;
int **puntero2 = &puntero1;

-----------------------
Existen en C los punteros función o function pointers que, en lugar de apuntar a una variable, apuntan a una función. Esto significa que almacena la dirección de una función, permitiéndote llamar a la función usando un puntero. Ejemplo:

int (*puntero) (int ,int);

En este caso estamos creando un puntero que guarda la dirección de memoria de una función que retorna un entero y tiene dos parámetros enteros. Podemos asignarle una función así:

puntero = sumar;

o

puntero = &sumar;

Siendo "sumar" una función que hemos creado. Para llamar a la función a través del puntero podemos hacer así:

int suma = puntero(12,56);

o

int suma = (*puntero)(12,56);

Nota: El nombre de una función en sí mismo apunta al comienzo de su código en memoria. Eso quiere decir que el nombre de una función actúa como un puntero; declarar un puntero función simplemente te da una variable que puede almacenar esa dirección.
------------------
En C se puede crear arrays de funciones, por ejemplo:

void (*saludos[3])() = {saludo,saludo2,saludo3};

En este ejemplo el array se llama saludos y almacena 3 funciones: saludo, saludo2 y saludo3. Para llamar, por ejemplo, a saludo2 a través del array se hace así:

saludos[1]();
