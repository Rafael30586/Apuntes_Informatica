Un servlet java es un programa java que corre en un servidor web habilitado para java o en un servidor de aplicación. Se encarga de las solicitudes del cliente, las procesa y genera respuestas dinámicamente. Los servlets son la columna vertebral de muchas aplicaciones java del lado del servidor debido a su eficiencia y escalabilidad.

------------------
Características de los servlets de java
* Funcionan del lado del servidor
* Se encargan eficientemente complejas solicitudes del cliente
* Generan respuestas dinámicas
* Ofrecen mejor rendimiento comparado a tecnologías más viejas como CGI
* Altamente escalables en aplicaciones web de nivel empresarial

------------------
Arquitectura de los java servlets
Los contenedores de los java servlet juegan un papel muy importante. Es responsable de encargarse de tareas importantes como balanceo de carga, gestión de sesión, asignación de recursos; se asegura de que todas las solicitudes se procecen eficientemente bajo elevado tráfico

Flujo de trabajo de la arquitectura de servlet
* Los clientes envían la solicitud al servidor web
* El servidor web recibe la solicitud
* El servidor web pasa la solicitud al servlet correspondiente
* El servlet procesa la solicitud y genera la respuesta en la forma de output
* El servlet envía la respuesta de vuelta al servidor web
* El servidor web envía la respuesta de vuelta al cliente y el browser del cliente la muestra en pantalla
