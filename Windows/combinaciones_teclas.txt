alt + tab --> navega entre diferentes ventanas de aplicaciones abiertas.

ctrl + tab --> navega entre diferentes pestańas de una aplicación abierta.

f6 --> cambia entre secciones internas

alt + space --> despliega menú para minimizar, maximizar etc.

ctrl + t --> abre una nueva pestańa en el navegador mientras lo estamos usando.

ctrl + w --> cierra la pestańa actual.

ctrl + <número> --> abre la pestańa que corresponda al número que indiquemos.

alt + f4 --> cierra la ventana actual

shift + click derecho --> abre un menú distinto al habitual
