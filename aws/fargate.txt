¿Qué es AWS Fargate?

AWS Fargate es un servicio completamente gestionado que te permite correr aplicaciones en contenedores en AWS sin tener que gestionar los servidores o clusters subyacentes. Ya sea que estés usando ECS o EKS, Fargate abstrae la gestión de infraestructura y te permite enfocarte en desarrollar y desplegar tus aplicaciones
Con Fargate, solo pagás por los recursos y contenedores que usás (CPU y memoria), y AWS aprovisiona automáticamente la infraestructura apropiada para vos. Esto hace que AWS Fargate de buen costo y eficiente para desarrolladores y negocios que estèn buscando simplificar la gestión de contenedores

---------------------
Características de AWS Fargate

* Gestión de contenedores serverless: No hay necesidad de gestionar servidores. Fargate se encarga de aprovisionar y escalar la infraestructura automáticamente
* Precio pay-as-you-go: Solo pagás por el cómputo y recursos de memoria usados por tus contenedores. Esto permite operaciones de mejor costo
* Integración sin problemas con ECS y EKS: Fargate funciona sin problemas con Amazon ECS y EKS, permitiéndote utilizar servicios de AWS familiares para orquestación de contenedores
* Escalado automático: Fargate ajusta automáticamente tus recursos basándose en las necesidades de tu aplicación, escalando para arriba o para abajo dependiendo del tráfico y la demanda
* Seguridad y cumplimiento: AWS Fargate se integra con servicios de AWS security como IAM y VPC para asegurar que tus aplicaciones estén seguras y tus datos protegidos
