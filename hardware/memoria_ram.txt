Random Access Memory

La RAM es la memoria principal de la computadora que almacena temporalmente datos y permite rápidas operaciones de escritura y lectura. Localizada en la placa madre, apoya directamente al CPU, permitiendo que los programas corran de manera fluida y que la información se procese rápidamente
* Los datos de la memoria RAM se pierden cuando la computadora se apaga
* Su alta velocidad influencia de manera importante al rendimiento del sistema, y RAM insuficiente puede ralentizar la carga de programas y reducir eficiencia.

Evolución histórica de la memoria RAM

1947 - Williams Tube - Primer tipo de RAM usando puntos cargados eléctricamente en tubos de rayos catódicos
1947 - Magnetic-core memory - RAM hecha de pequeños anillos de metal para almacenar 1 bit de datos, fácilmente accesible
1968 - DRAM (RAM dinámica) - Inventada por Robert Dennard, usando transistores para guardar datos, requiere energía
1969 - Intel 1103 DRAM - Primer producto DRAM de Intel, marcando el uso comercial de la tecnología DRAM
1993 - SDRAM - Samsung lanzó KM48SL2000, DRAM síncrona para procesamiento más rápido
1996 - DDR SDRAM - Se volvió comercialmente disponible, ofreciendo velocidades más altas
1999 - RDRAM - Lanzado para computadoras con un enfoque en tasas más rápidas de datos
2003 - DDR2 SDRAM - Lanzada con mejor velocidad y eficiencia
2007 - DDR3 SDRAM - Ofreció velocidades más altas y mejor eficiencia energética
2014 - DDR4 SDRAM - Disponible con velocidades aún más altas y mayor eficiencia energética

Funcionamiento de la RAM

La RAM está hecha de pequeños transistores y capacitores que almacenan cargas eléctricas representando bits de datos. Funciona como la memoria a corto plazo de la computadora, temporalmente reteniendo datos a los cuales el CPU necesita acceder rápidamente

Como la RAM funciona en la práctica
Al correr programas, la computadora carga datos del almacenamiento a la RAM para que el CPU pueda acceder de manera rápida. Más RAM y memoria más rápida le permite a tu sistema encargarse de archivos grandes, hacer multitareas más eficientemente y correr aplicaciones demandantes de manera más fluida.
Más RAM le permite a tu computadora hacer malabares con más tareas a la vez y procesar datos más rápido

-------------------------
RAM vs otros tipos de memoria

* RAM vs disco duro: la RAM es rápida, memoria temporal usada para tareas activas, mientras que un disco duro provee almacenamiento permanente y más lento para archivos y programas
* RAM vs ROM: la RAM es volátil y escribible, almacenando datos temporalmente, mientras la ROM es no volátil y almacena instrucciones permanentes que no pueden ser modificadas fácilmente
* RAM vs memoria virtual: la RAM es memoria física para procesamiento inmediato, mientras que la memoria virtual usa una porción del disco duro como RAM temporal cuando la memoria física está llena, pero es mucho más lenta
